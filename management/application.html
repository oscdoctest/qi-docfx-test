CreateClientApiKeyForApplication()

Create a ClientApiKeySet <ClientApiKeySetObj> for the specified application.

Http

    POST api/Tenants/{tenantId}/ClientApiKeySets/Keys

Parameters

String tenantId ID of the tenant the application belongs to. String applicationId ID of the application for this request String description Description of the :ref:ClientApiKeySet <ClientApiKeySetObj>

Security Allowed by Account Administrator :ref:Role <RoleObj>

Returns The created :ref:ClientApiKeySet <ClientApiKeySetObj>

Notes For HTTP requests and responses, the JSON serialized ClientApiKeySet object Looks like:

.. _ClientApiKeySetObj:

.. highlight:: C#

    {
        "AppUri": "appuri",
        "CreateFirstKey": false,
        "DisplayName": "displayname",
        "RequiredResource":  { },
        "TenantId": "tenantid"
    }
